Mercenary deterrence: TVL to add before the flip

Once emissions land, the gauge's displayed vAPR = annualized emissions ÷ staked TVL. A high vAPR attracts mercenary LPs who dilute our share. Adding TVL before the epoch flip keeps the displayed vAPR below the target ceiling so the pool never shows up on farming radars.

Assumptions
  • Emissions to a gauge = total gauge emissions × pool votes / total votes. Only 45% of Minter weekly (8,969,150 AERO) reaches gauges — the veAERO rebase and team allocation are carved out — so the total is measured by summing every live gauge's streaming rate (4,031,192 AERO/wk).
  • One-epoch lag: votes cast now direct emissions that start streaming at the next flip. The gauge currently streams 2,192 AERO/wk (set by last epoch's votes); current votes imply 2,173 AERO/wk at the next flip.
  • Total votes are held constant: reallocating our votes moves them between pools without changing the total.
  • Own-pool fees+bribes use the latest epoch's $10.74 (trailing 4-epoch average $7.34); voting more for our own pool doesn't create new fees, it only changes our share of them.
  • Other-pool voting income assumes the freed/remaining votes stay spread across our current pools in the same proportions.
  • Incumbent staked TVL is assumed static; in practice mercenary TVL chases displayed vAPR — that response is exactly what the deterrence calculator is for.
  • Prices are point-in-time (AERO $0.5104, kVCM $0.0325); no price-impact modeling for acquiring kVCM or for AERO sell pressure from farming.
  • LP deposits into a vAMM pool must be balanced 50/50 by value at deposit time, so the kVCM/USDC split cannot be chosen: half the deposit's value must be provided as each token (the scenario shows the implied amounts at current prices).
  • Cross-sectional hurdle: emissions-vAPR = gauge streaming rate × 52.14 weeks × AERO price ÷ staked TVL, over live gauges streaming emissions; tokens priced via Alchemy spot with prices.csv fallback, gauges with unpriceable tokens skipped. Pool "type" from LpSugar (≤ 0 = basic AMM, > 0 = CL).
  • Historical ceiling: pool-history.csv snapshots gauge state 1h after each epoch flip via archive calls; the AERO rate per epoch comes from the gauge's own rewardRateByEpoch record. Our TVL is defined as the gauge balance of the voter address only — LP staked by any other address counts as external, including addresses we might control that are not the voter.
  • The inferred ceiling defaults to the stricter of (cross-sectional p25, lowest vAPR that ever drew external inflow), rounded and clamped to 5–50%; it is a starting point, not a hard rule.
